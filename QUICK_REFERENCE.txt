Problem: Categories not showing in frontend
Solution:
  1. Check backend is running: http://localhost:8083/api/jobs/categories
  2. Check browser console for errors
  3. Verify database has categories: SELECT COUNT(*) FROM job_categories;

Problem: Can't add job with category
Solution:
  1. Check category exists: SELECT * FROM job_categories WHERE category_id = X;
  2. Check foreign key constraint is enabled
  3. Verify user has permission to create jobs

Problem: Foreign key constraint error
Solution:
  1. Check category_id exists in job_categories table
  2. Make sure category is not deleted
  3. Use valid category_id when creating job
